Folder | Source | What it is
iem_asos_1min | Iowa Environmental Mesonet (IEM) | ASOS 1-minute temperature data (~24h delay)
awc_metar | Aviation Weather Center (AWC) | METAR observations (hourly + SPECI)
iem_daily_climate | Iowa Environmental Mesonet (IEM) | NWS Daily Climate Report (CLI) - official high/low for Kalshi settlement. Chicago=CLIMDW (Midway). See docs/events/kalshi_settlement_rules.md
kalshi_market_snapshots | Kalshi WebSocket | Market snapshots (yes_bid, yes_ask, no_bid, no_ask, last_price)
kalshi_orderbook_snapshots | Kalshi WebSocket | Orderbook depth
synoptic_weather_observations | Synoptic WebSocket | Real-time weather observations
weather_bot_paper_trades | Paper execution | Bot trade log (CSV + parquet YYYY-MM-DD.parquet)
kalshi_historical | Kalshi REST API | Historical candlesticks and trades (backfill)
hrrr | NOAA HRRR via AWS S3 (noaa-hrrr-bdp-pcs) | HRRR 15-min sub-hourly 3km point forecasts (2m temp) at station coordinates
rtma_ru | NOAA RTMA-RU via AWS S3 (noaa-rtma-pds) | RTMA Rapid Update 2.5km analysis (2m temp) at station coordinates
rrfs | NOAA RRFS via AWS S3 (noaa-rrfs-pds) | RRFS 3km hourly point forecasts (2m temp) at station coordinates (prototype, paused Dec 2024)
nbm | NOAA NBM via AWS S3 (noaa-nbm-grib2-pds) | NBM 2.5km bias-corrected hourly point forecasts (2m temp) at station coordinates

Download scripts: research/download_data/

  rtma_ru.py - RTMA Rapid Update analysis fetcher

  run_nwp_collection.py - Unified NWP script for all models (latest + backfill)
  run_hrrr_collection.py - HRRR-specific convenience script
